# Run Grid

```bash
$ java -jar selenium-server-standalone-3.2.0.jar -role hub -hubConfig hub_config.json
```

# Run Appium

```bash
$ node . --nodeconfig ~/GitHub/ruby_lib/grid/config.json

# or

$ appium --nodeconfig ~/GitHub/ruby_lib/grid/config.json
```

# Run Ruby Script

```bash
$ rake ios # for example
```
